Reddit Memestic Bot API Documentation

Tài liệu các endpoint API cho bot Reddit Memestic. Các endpoint đều trả về JSON (trừ trang home và /users).

POST /send-now

Gửi meme mới (ảnh từ Reddit, caption là tiêu đề Reddit) đến Telegram.

Request body: 
{}
Ví dụ curl: 
curl -X POST http://localhost:4000/send-now -H "Content-Type: application/json" -d '{}'
Response: 
{
  "success": true,
  "type": "photo",
  "usedImageUrl": "url_ảnh",
  "caption": "caption"
}

POST /send-now-text

Gửi text đến Telegram.

Request body: 
{
  "text": "Nội dung gửi"
}
Ví dụ curl: 
curl -X POST http://localhost:4000/send-now-text -H "Content-Type: application/json" -d '{"text":"Hello"}'
Response: 
{
  "success": true,
  "type": "text",
  "sent": "Nội dung gửi"
}

POST /send-now-photo

Gửi ảnh đến Telegram (truyền url hoặc lấy meme mới).

Request body: 
{
  "photo": "url_ảnh",
  "caption": "caption tuỳ chọn"
}
Ví dụ curl: 
curl -X POST http://localhost:4000/send-now-photo -H "Content-Type: application/json" -d '{"photo":"https://i.imgur.com/your-image.jpg","caption":"Caption tuỳ chọn"}'
Nếu không truyền photo, server sẽ tự lấy meme mới. Response: 
{
  "success": true,
  "type": "photo",
  "usedImageUrl": "url_ảnh",
  "caption": "caption"
}

POST /send-now-preview

Nếu không truyền photo, server sẽ tự lấy meme mới. Response: 
{
  "success": true,
  "type": "preview",
  "usedImageUrl": "url_ảnh",
  "caption": "caption"
}
